EC2 인스턴스에서 액세스 키와 시크릿 키를 사용하지 않고 S3에 접근할 수 있도록  
IAM 역할을 생성하자.   
  
1. IAM 역할 이름을 설정한다.  
2. Amazone EC2를 선택한다.  
3. S3 버킷에서 파일을 받을 수 있어야 하기 때문에 Amazone S3 Full Access 권한  
을 선택한다.   
4. IAM 역할을 생성한다.
