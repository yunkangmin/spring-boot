## 코틀린 ?
https://philosopher-chan.tistory.com/58

```
코틀린에서 null 처리에 대해서 좀 예민한데
코틀린에서 ?는 간단하다
?는 null일 수 있음을 의미한다.
그러니깐
var abc:String? = null
을 보면 abc라는 변수가 null일 수도 있다 이기 때문에, null을 집어넣어도 에러가 나지 않는다.
그렇다면 !!는 뭘까요?
value!!.value 처럼 쓰이는데
!!는 null이 아니다 라고 선언해주는 것입니다.
이렇게 간단하게 !!와 ?에 대해서 정리해봤습니다.
```
